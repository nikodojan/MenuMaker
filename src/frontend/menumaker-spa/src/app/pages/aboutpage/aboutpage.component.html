About MenuMaker

At the moment, the Web App is in development, and runs on free tiers in Azure, to save development costs. For local testing, it is deployed to a microk8s cluster, with a PostgreSQL database on Ubuntu.

While the WebApp is deployed as a Static WebApp and is permanently accessible, the API needs some time to start up after being paused.

Api docs

An API documentation can be found at https://menumakerapi.azurewebsites.net/swagger/index.html.
At the moment there are only save HTTP methods are accessible.
