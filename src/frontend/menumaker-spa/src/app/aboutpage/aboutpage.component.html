About MenuMaker

At the moment, the Web App is in development, and runs on free tiers in Azure, to save development costs. For local testing, it is deployed to a microk8s cluster, with a PostgreSQL database on Ubuntu.

In this online version, application and database are running on free tiers in Azure. Especially the database is therefore quite slow at cold start.

While the WebApp is deployed as a Static WebApp and is permanently accessible, the API and the Azure SQL database each need some time to start up after bein paused.

Api docs

An API documentation can be found at https://menumakerapi.azurewebsites.net/swagger/index.html.
At the moment there are only save HTTP methods are accessible. (Again, database could take a while to start up.)
